# Obsidian Scroll Control

<div align="center">
    <img src="assets/logo.svg" alt="Obsidian Scroll Control Logo" width="200"/>
</div>

A plugin for Obsidian that adds customizable scroll controls to enhance document navigation.

## Features

### Button Controls

-   Configurable floating scroll buttons
-   Multiple position options: floating, ribbon, or toolbar
-   Adjustable horizontal and vertical padding
-   Auto-hide functionality with configurable delay

### Visual Design

-   Clean SVG icons
-   Configurable button sizes (small, medium, large)
-   Custom color support
-   Smooth transitions

### Configuration Options

-   Button visibility toggles
-   Animation speed control
-   Adjustable scroll step size
-   Position customization (0-100px horizontal, 20-100px vertical)

## Installation

1. Open Obsidian Settings
2. Navigate to Community Plugins and disable Safe Mode
3. Click Browse and search for "Scroll Control"
4. Install and enable the plugin

## Usage

### Basic Configuration

1. Enable the plugin in Community Plugins
2. Select desired scroll buttons (Top, Bottom, Up, Down)
3. Choose button position preference

### Advanced Settings

-   Button positioning and padding
-   Visual appearance (size, color, animations)
-   Scroll behavior and timing
-   Auto-hide preferences
